Add Modal Register and login to WordPress, 为 WordPress 添加模式窗口注册登录功能。

== Description ==

WordPress 模态窗口注册、登录、找回密码插件。

- 支持微博、微信、QQ 登录
- 支持找回密码
- 在模态窗口中切换注册、登录、找回密码
- 注册登录模板支持自定义
- 前段样式依赖 wizhi cms 或 pure css


= 支持功能

* WordPress 标准的注册登录
* 注册时显示验证码
* QQ、微博、微信登录


作者网址：[WordPress智库](http://www.wpzhiku.com/ "WordPress CMS 插件")


== Installation ==

1. 上传插件到`/wp-content/plugins/` 目录
2. 在插件管理菜单激活插件
